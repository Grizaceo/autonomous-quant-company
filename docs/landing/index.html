Why this beats prompt trading

Prompt trading asks a language model to improvise a trade. AQTC does the opposite: it starts with offline evolved alpha, freezes the evidence, compares it against a bad rejected strategy, and only then lets agents operate inside a policy-bounded business loop.

  • Frozen provenance: SHA-256 proof manifest and demo artifacts.
  • Risk gate: no live broker execution by default.
  • Public contracts: CLI, API and MCP outputs are test-locked.
  • Revenue proof path: Stripe test mode optional, mock ledger deterministic.
